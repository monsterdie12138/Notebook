    Chapter1 概述

    1.1 c++面向对象程序设计

    OOP：

    面向对象编程（Object Oriented Programming ，简称OOP） 是一种编程思想，它将程序看作一个由各种独立的对象（object） 组成的集合。每个对象都有自己的属性和方法，并且可以与其他对象进行交互，共同完成任务。在面向对象编程中，通过类（class） 和实例（instance） 来描述具体事物，类定义了一类事物的通用属性和方法，而实例则是具体某个事物的一个具体表现。

    特性：

    • Abstraction 抽象
    • Encapsulation 封装
    • Inheritance 继承
    • Polymorphism 多态

    1.2 c++

    标准库分类：

    • C标准库： 强调面向过程
    • C++标准库： 包括C++I/O库、String库、时间库、异常库、数值库，提供基本的输入/输出、字符串处理和数值计算功能
    • 标准数据库STL： 包括容器（Containers）、算法（Algorithms）、迭代器（Iterators）、函数对象（Function Objects）、配接器（Adapters）、分配器（Allocators）

    数据类型：

    • 基本类型： int, char, long, wchar_t, char16_t, char32_t
    • 实型： float, double, long double
    • 逻辑类型和空类型： bool, void
    • 自定义类型： struct, class, union, enum, pointer, array
    • STL中的类型： vector, string, list, stack, map, set
    • c++11： long long

    字符串：

    定义与初始化：

    string c; //定义空字符串
string c1("this is a string"); //用指定字符串初始化

string s[10]; //定义限制长度的字符串
string s(5,’c’); //设置参数，用5个c初始化

    string s1("I am boy"),s3;

    1.3 数据输入与输出

    stream 流：

    多数情况下的流是从源设备到目标设备的字符序列。

    • istream 输入流： 从输入设备流向内存的字节序列 对应于c++的cin
    • ostream 输出流： 从内存流向输出设备的字节序列 对应于c++的cout

    cin和提取运算符>>:

    输入单个变量的值：

    cin>>x;

    输入多个变量的值：

    cin>>x1>>x2>>x3...>>xn;

    Note

    cin具有自动识别数据类型的能力； 在>>后面只能出现变量名，下面的语句是错误的： 

    cin>>"x=">>x;
cin>>12>>x;
cin>>'x'>>x;

    cout和插入运算符<<:

    输出单个变量的值：

    cout<<x;

    输出多个变量的值：

    cout<<x1<<x2<<x3<<...;

    输出换行：

    cout<<endl;
cout<<"\n";

    c++常用转义字符：

    名称 字符形式 值
    空字符(Null) \0 0X00
    换行(NewLine) \n 0X0A
    换页(FormFeed) \f 0X0C
    回车(Carriage Return) \r 0X0D
    退格(BackSpasc) \b 0X08
    响铃(Bell) \a 0X07
    水平制表(Horizontal Tab) \t 0X09
    垂直制表(Vertical Tab) \v 0X0B
    反斜杠(backslash) \\ 0X5C
    问号(question mark) \? 0X3F
    单引号(single quote) \' 0X27
    双引号(double quote) \" 0X22
    控制符 功能
    endl 输出一个新行符，并清空流
    ends 输出一个空格符，并清空流
    dec 用十进制表示法输入或输出数值
    hex 用十六进制表示法输入或输出数值
    oct 用八进制表示法输入或输出数值
    setfill(char c) 设置填充符c
    setprecision (int n) 设置浮点数输出精度(包括小数点)
    setw(int n) 设置输出宽度

    输出格式控制符：

    setprecision(n)：保留n位有效位，四舍五入。

    cout<<setprecision(3)<<3.14126<<" "<<2.4576<<endl;

输出：3.14 2.46

    setw(n)：设置输出的宽度，其只对紧接着的输出产生作用。

    当输出字段长度小于n时，在该字段之前用空格补齐；当输出字段大于n时，整体输出，没有变动。

    cout<<setw(4)<<123456<<endl;

输出：123456

    cout<<setw(8)<<123456<<endl;

输出：__123456（_表示空格）

    setiosflags(long f)和resetiosflag(long f)：设置或重置流的格式。

    cout<<setiosflags(ios::left)<<setw(8)<<123<<setw(8)<<456<<endl; 
cout<<resetiosflags(ios::left)<<setw(8)<<123<<setw(8)<<456<<endl;

123_____456_____
_____123_____456

    cout<<setiosflags(ios::uppercase)<<hex<<255<<endl;
cout<<resetiosflags(ios::uppercase)<<hex<<255<<endl;

输出：
FF
ff

    数制基数：

    数制基数在设置后将一致有效，直到下一次设置新的数制基数才取消。

    cout<<hex<<17<<" "<<17<<endl;
cout<<17<<" "<<dec<<17<<endl;
cout<<17<<" "<<oct<<17<<endl;

输出：
11 17
11 17
17 21
